=head1 NAME

ob-version - print out version of g-speak and other system components

=head1 SYNOPSIS

B<ob-version> S<[ B<-y> ]>

=head1 DESCRIPTION

B<ob-version> prints out the version number of the g-speak
installation which it is a part of.  It also collects some other basic
version and configuration information about the system and prints that
out as well.

=head1 OPTIONS

=over 4

=item B<-y>

Instead of printing the information in a human-readable format, print
it in JSON format.  Since YAML is a superset of JSON, it can also be
read by any compliant YAML parser, hence the 'y'.  Beware that Syck
(used by Ruby's YAML implementation) is not a compliant YAML parser,
and it may have issues reading escaped characters in the JSON data.
(Though escaped characters are likely to be rare in the strings
generated by ob-version.)  For this reason, JSON.load is recommended
over YAML.load when parsing the output in Ruby.

=back
